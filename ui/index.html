Why isn't it working ?

Make sure you're properly connected to the internet and using the most current version of TERA Toolbox

Did you try "u reset" command and the issue persists ? Did you check for conflicting mods ?

If the issue persists, try posting in #help

Why can't I open the in-game UI ?

If you can't open the cash shop store in-game, it's an issue with a lost or deleted file (Awesomium in "client/Binaries") Try a repair tool to get it back

Why can't I see my/my friends' accessories/hair/face/etc ?

Most likely because you or your friends using a conflicting mod. Again, conflicting mods are as follows: Risenio FPS Utils, FPS Booster, PinkiePie's Datacenter, Arborean Apparel, Cosplayer, Surgeon, Quickload, No More Crazy Capes

When I use my Marrow/Dracoloth brooch, my face and voice resets ?

In config, toggle on "Ignore brooch transformations"

Where are my character profiles saved ?

Accessible from "Proxy\mods\unicast\saves" and can be moved at will

Why isn't the UI opening ?

1. Make sure both Unicast and Toolbox are working properly in your console

2. Try resetting your UI position in game settings

3. Check to make sure your awesomium is intact and working. (If your TERA store will open, it is)

Why can't I see footsteps ?

You have to enable Travel Effects in your base game. This doesn't have to do with Unicast

When will Unicast get XXX costume ?

When it exists in your region's client. Meaning, we can't put files that don't exist into your game, it doesn't work like that

Usage

Most functions can be accessed from the UI

Prefix: u (ex: u reset, u ui)

Always use commands in /8 Proxy channel:

u ui -> opens ui (alternatively use "ctrl + shift + u")

u reset slot -> resets a slot (resets all if no slot is specified - ex: u reset body, u reset)

u equip itemId -> equips an item by id

u toggle -> toggles mod on or off

u connect -> connects to the costume server

u disconnect -> disconnects from the costume server

u refresh -> refreshes other mod user's appearance

Themes

Select your custom or pre-made theme from the config window ! Change them freely. To create your own, navigate to the "unicast\ui\css\themes" folder to get started

Race & Idle Change

- You can only Race Change if the race+gender combo allows it. (Ex: Elin Priest can become any race+gender, Valkryies can only be castanic female)

- Idles can only be swapped if there is an alternative idle. (Ex: Elin Priest can use Ninja, Brawler, or Default idle, Valkyries can only use the Default castanic female idle)

- Idles will temporarily revert when a skill is used. (This is to prevent desync in combat)

- Unicast selected idles will resume after some time out of combat. (To speed up the transformation, talk to an npc or use an emote)
